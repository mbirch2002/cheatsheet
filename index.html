CSS selectors/specificity
Type Specificity Example
Inline styles 1000 <h1 style="color: pink;"></h1>
IDs 100 #navbar
Classes, pseudo-classes, attribute selectors 10 .test, :hover, [href]
Elements and pseudo-elements 1 h1, ::before

Note: Inline style gets a specificity value of 1000, and is always given the highest priority!
Note 2: There is one exception to this rule: if you use the !important rule, it will even override inline styles!
